This demo shows you how to use an event to start/stop tagging.
On top of that we disable onclick tag so you have complete control on when the user can tag

$("#img1").tag({
	clickToTag: false
});
$("#enable").click(function(){
	$("#img1").showDrag();
});
$("#disable").click(function(){
	$("#img1").hideDrag();
});
